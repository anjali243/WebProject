# School Website

Live Website: https://anjali243.github.io/WebProject/

## Overview

This project is a static school website created to provide information about the school, including academics, admissions, events, faculty, and contact details.

The website is hosted using GitHub Pages.

## Features

- Responsive homepage
- About school section
- Admission information
- Faculty details
- Events & announcements
- Contact page
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript
